# Stepy::Rails

This is the stepy jquery plugin packaged as a gem. 
See the web site: http://wbotelhos.com/stepy
## Installation

Add this line to your application's Gemfile:

    gem 'stepy-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install stepy-rails
